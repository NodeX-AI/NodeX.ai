# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ телеграм-бота NodeX.ai

* **Сбор данных.** Бот хранит только необходимые для работы данные: телеграм ID пользователя, текст сообщений и ответов моделей, выбранная модель, временные метки.
* **Защита данных.** Все сообщения шифруются алгоритмом AES-256-GCM перед сохранением в базу данных. Ключ шифрования хранится отдельно от базы данных.
* **Использование данных.** Данные используются исключительно для: предоставления услуг модели, улучшения качества ответов, технической аналитики.
* **Публичность данных.** Разработчик не передает данные третьим лицам, не использует данные для тренировки моделей, не публикует переписку пользователей.
* **Удаление данных.** Пользователь имеет возможность: удалить историю переписки с моделью, удалить аккаунт. Удалить данные можно при помощи кнопки в меню "Опасная зона".
* **Согласие с политикой.** Начиная использование бота через команду /start, вы автоматически соглашаетесь с: данной Политикой Конфиденциальности, политикой Ограничения Ответственности, Условиями использования сервиса.
* **Вы можете в любой момент отозвать согласие и удалить свой все свои данные, при условии удаления аккаунта. При удалении аккаунта безвозвратно удаляются: ваши запросы и ответы модели вам, персональные данные (Telegram ID), история взаимодействия с ботом.**
* **Фотографии.** При отправке изображений боту вы соглашаетесь с тем, что:
    - Изображение будет временно загружено на сторонний бесплатный хостинг (freeimage.host)
    - URL вашего изображения будет передан сервису OpenRouter.ai для обработки нейросетью
    - Бот не сохраняет ваши изображения на серверах
    - Изображение удаляется с хостинга автоматически через некоторое время
    
Разработчик не рекомендует отправлять конфиденциальные, личные или коммерческие изображения
