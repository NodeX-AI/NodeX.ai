# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ телеграм-бота NodeX.ai

* **Сбор данных:** бот хранит только необходимые для работы данные: телеграм ID пользователя, текст сообщений и ответов моделей, выбранная модель, временные метки,а также изображения, загружаемые для распознавания (далее — «Изображения»). Изображения хранятся в защищенном хранилище S3, остальные данные сохраняются в базу данных.
* **Правовые основания и цели обработки:** ваши данные обрабатываются на основании вашего добровольного согласия, которое вы выражаете явным действием перед началом использования бота. Цели обработки: предоставление услуг модели, улучшение качества ответов, техническая аналитика.
* **Защита данных:** все сообщения шифруются алгоритмом AES-256-GCM перед сохранением в базу данных. Ключ шифрования хранится отдельно от базы данных. Для обеспечения работы сервиса данные хранятся на защищенных серверах хостинг-провайдера Timeweb Cloud на территории Российской Федерации.
* **Использование данных:** данные используются исключительно для заявленных целей: предоставление услуг модели, улучшение качества ответов, техническая аналитика.
* **Публичность данных:** разработчик не использует данные для тренировки моделей, не публикует переписку пользователей. Для технического обеспечения работы бота (хостинг) данные передаются хостинг-провайдеру Timeweb Cloud.
* **Обработка изображений.** Для реализации функции распознавания изображений загружаемые файлы временно размещаются в защищенном объектном хранилище (S3), предоставляемом Timeweb Cloud. Ссылки на эти изображения передаются моделям искусственного интеллекта для анализа. Исходные файлы удаляются из хранилища сразу после завершения обработки запроса.

## Ваши права:

Помимо удаления данных, вы имеете право:

* Получить информацию о том, какие ваши данные обрабатываются.
* Потребовать уточнить ваши данные, если они неполны или устарели.
* Вы можете реализовать эти права, обратившись к разработчику через почту nodex.project@gmail.com.

* **Удаление данных:** пользователь имеет возможность удалить историю переписки с моделью или полностью удалить аккаунт. Удалить данные можно при помощи кнопки в меню "Опасная зона". Обработка данных прекращается, а данные удаляются из базы данных безвозвратно в течение 7 дней с момента удаления аккаунта.

## Согласие с политикой

Начиная использование бота, вы принимаете соглашение на обработку данных. Настоящая Политика является неотъемлемой частью Пользовательского соглашения.

## Изменения в политике

Разработчик может обновлять данную Политику. Актуальная версия всегда будет доступна здесь. При внесении существенных изменений мы уведомим вас через бота.
